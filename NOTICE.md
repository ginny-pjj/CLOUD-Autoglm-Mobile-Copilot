本项目基于 [zai-org/Open-AutoGLM](https://github.com/zai-org/Open-AutoGLM) 构建。

Open-AutoGLM 提供 Phone Agent 核心能力。本仓库在其上增加了：

- FastAPI 任务服务（`server/`）

- Mock / Real 双模式与结构化 Trace
- Docker 云端部署与 Tailscale 远程 ADB 方案
- 远程场景优化（截图超时、Take_over、美团快路径等）
